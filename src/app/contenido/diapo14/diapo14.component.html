What will happen after issuing the command “access-group 115 in” on the s0/0/1 interface?

Solution:

Hosts cannot connect to Router through s0/0/1.

Recall that each interface only accepts one access-list, so when using the command “ip access-group 115 in” on the s0/0/1 interface it will overwrite the initial access-list 102. Therefore any telnet connection will be accepted
