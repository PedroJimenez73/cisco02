Each of these four switches has been configured with a hostname, as well as being configured to run RSTP. No other configuration changes have been made. Which is the correct RSTP port roles for the indicated switches and interfaces?

Solution:

SwitchD ➡︎ Gi0/2 root

SwitchA ➡︎ Fa0/2 designated

SwitchD ➡︎ Fa0/1 root
