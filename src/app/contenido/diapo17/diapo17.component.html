Which command is used to find what kind of router is VLAN-R1?

Solution:

VLAN-R1 is the router directly connected to Sw-Ac3 switch, so we can use the show cdp neighbors command.


This command permit to see:


  1. Neighbor Device ID : The name of the neighbor device;
  2. Local Interface : The interface to which this neighbor is heard
  3. Capability: Capability of this neighboring device – R for router, S for switch, H for Host
  4. Platform: Which type of device the neighbor is
  5. Port ID: The interface of the remote neighbor you receive CDP information
  6. Holdtime: Decremental hold time in seconds
