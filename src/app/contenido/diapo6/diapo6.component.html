Solution:

  • Native VLAN mismatch between SW1 & SW3
  • SW2 interface Ethernet0/2: Wrong VLAN assignation.
  • Switchport mode mismatch in SW1.

Recommended commands: show interfaces trunk, show vlan, show running-config.

Commands to fix the issues:


1. Fix the native VLAN issue on SW3:


Sw3(config)# interface Ethernet0/0

Sw3(config-if)# switchport trunk native vlan 1


2. Fix the issue on SW2:


Sw2(config)# interface Ethernet0/2

Sw2(config-if)# switchport access vlan 500


3. Fix the issue on SW1:


Sw1(config)# interface Ethernet0/0

Sw1(config-if)# switchport trunk encapsulation dot1q

Sw1(config-if)# switchport mode trunk
