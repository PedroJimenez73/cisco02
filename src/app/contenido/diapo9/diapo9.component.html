8Refer to the exhibit. A network engineer must block access for all computers on VLAN 20 to the web server via HTTP. All other computers must be able to access the web server. Which configuration when applied to switch A accomplishes this task?

Solution:

config t

ip access-list extended wwwblock

deny tcp any host 10.30.0.100 eq 80

permit ip any any

int vlan 20

ip access-group wwwblock in
