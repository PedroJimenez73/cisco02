9Refer to the exhibit. An extended ACL has been configured and applied to router R2. The configuration failed to work as intended. Which two changes stop outbound traffic on TCP ports 25 and 80 to 10.0.20.0/26 from the 10.0.10.0/26 subnet while still allowing all other traffic?

R2#config t

R2(config)#access-list 101 deny tcp 10.0.20.0 0.0.0.63 10.0.10.0 0.0.0.63 eq smtp

R2(config)#access-list 101 deny tcp 10.0.20.0 0.0.0.63 10.0.10.0 0.0.0.63 eq www

R2(config)#int gi0/2

R2(config-if)#ip access-group 101 in

Solution:

Add a “permit ip any any” statement at the end of ACL 101 for allowed traffic

The source and destination IPs must be swapped in ACL 101
